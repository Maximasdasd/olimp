# АРМ «Олимпиадное движение»

Автоматизированное рабочее место для ввода, хранения и обработки информации
об участии студентов в олимпиадном движении и конкурсах профессионального
мастерства.

Курсовая работа по дисциплине **«МДК 05.03 Тестирование информационных систем»**
(ГАПОУ СО «Екатеринбургский монтажный колледж»).

## Архитектура

Единая система из трёх слоёв: общая база **PostgreSQL**, бэкенд на **FastAPI**
и фронтенд на **Django**, который обращается к бэкенду по HTTP.

```
┌────────────┐      ┌─────────────────────┐      ┌──────────────────────┐
│ PostgreSQL │ ◀──▶ │  FastAPI (бэкенд)    │ ◀──▶ │  Django (фронтенд)    │ ◀──▶ браузер
│  olimpiad  │      │  olimp_api           │ HTTP │  olimp_django         │
└────────────┘      │  REST API + JWT      │ JWT  │  прокси к FastAPI      │
                    └─────────────────────┘      └──────────────────────┘
```

- **PostgreSQL** — единое хранилище данных.
- **FastAPI (`olimp_api`)** — бэкенд: вся бизнес-логика, доступ к БД, REST API,
  аутентификация (JWT). Архитектура MVC.
- **Django (`olimp_django`)** — фронтенд: HTML-страницы и формы. Своих данных не
  хранит, за всё обращается к FastAPI. Вход выполняется через FastAPI, JWT-токен
  хранится в сессии Django.

| Часть | Папка | Технология | Роль |
|-------|-------|-----------|------|
| Бэкенд | [`olimp_api/`](olimp_api/) | FastAPI + SQLAlchemy + PostgreSQL | REST API, БД, логика |
| Фронтенд | [`olimp_django/`](olimp_django/) | Django + requests | Веб-интерфейс (прокси к API) |

## Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Незарегистрированный** | Просмотр списка олимпиад, фильтр по году, протоколы победителей, положения, задания прошлых лет |
| **Студент** | Запись на олимпиады, просмотр своих результатов, профиль |
| **Преподаватель** | Создание учётных записей студентов, положения и задания, ведение протоколов и результатов, подготовка/отзыв протоколов, бланки сертификатов |
| **Администратор** | Создание/редактирование учётных записей преподавателей, карточки олимпиад, публикация подготовленных протоколов, годовой отчёт |

**Жизненный цикл протокола:** `формируется` → `подготовлен` → `опубликован`.
После публикации протокол виден незарегистрированным пользователям. Отозвать
можно только протокол со статусом «подготовлен».

## Аутентификация

Вход — **по логину и паролю**. Единый источник учётных записей — FastAPI:

- В **FastAPI** (`/docs`) — OAuth2 password flow, кнопка **Authorize**.
- В **Django** — форма входа, которая отправляет логин/пароль в FastAPI и
  сохраняет полученный JWT в сессии. Токены вручную вставлять не нужно.

## Быстрый старт (локально, без Docker)

### 1. PostgreSQL

```bash
sudo -u postgres psql -c "ALTER USER postgres WITH PASSWORD 'root';"
sudo -u postgres psql -c "CREATE DATABASE olimpiad;"
```

### 2. Бэкенд (FastAPI)

```bash
cd olimp_api
pip install -r requirements.txt
cp .env.example .env                 # DATABASE_URL уже указывает на PostgreSQL
python -m scripts.seed_data          # демо-данные и тестовые учётки
uvicorn app.main:app --reload        # http://127.0.0.1:8000  (/docs — Swagger)
```

### 3. Фронтенд (Django)

В отдельном терминале (бэкенд должен быть запущен):

```bash
cd olimp_django
pip install -r requirements.txt
python manage.py migrate             # таблицы сессий Django
python manage.py runserver 8080      # http://127.0.0.1:8080
```

Откройте http://127.0.0.1:8080/ и войдите.

## Структура репозитория

```
olimp/
├── README.md            # этот файл
├── asd.pdf              # техническое задание
├── olimp_api/           # БЭКЕНД — FastAPI + PostgreSQL
│   ├── app/
│   │   ├── models/      # ORM-модели (Model)
│   │   ├── schemas/     # Pydantic-схемы (Model)
│   │   ├── controllers/ # бизнес-логика (Controller)
│   │   ├── routers/     # REST-эндпоинты (View)
│   │   ├── dependencies/# аутентификация, доступ по ролям
│   │   └── utils/       # конфигурация, безопасность
│   ├── scripts/         # seed_data.py — демо-данные
│   └── tests/           # тесты (pytest)
└── olimp_django/        # ФРОНТЕНД — Django (прокси к FastAPI)
    ├── config/          # настройки проекта
    ├── core/
    │   ├── fastapi_client.py  # клиент к FastAPI (JWT в сессии)
    │   ├── views.py           # вьюхи-прокси
    │   ├── decorators.py      # контроль доступа по ролям
    │   └── templates/         # HTML-шаблоны
    └── tests/           # тесты (pytest, FastAPI замокан)
```
