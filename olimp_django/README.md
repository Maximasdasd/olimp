# Олимпиадное движение — Web-приложение (Django)

Вторая часть курсовой работы: web-приложение на Django для обработки информации
об участии студентов в олимпиадном движении и конкурсах профессионального
мастерства. Дополняет REST-API из соседней папки `olimp_api`.

## Роли пользователей (по ТЗ)

- **Незарегистрированный пользователь** — просмотр списка олимпиад, фильтр по году,
  просмотр протоколов победителей (опубликованных), положений и заданий прошлых лет.
- **Студент** (зарегистрированный) — запись на олимпиады, просмотр своих результатов,
  редактирование профиля.
- **Преподаватель** — создание учётных записей студентов, прикрепление положений и
  заданий, создание/редактирование протоколов и результатов, подготовка и отзыв
  протоколов, добавление бланков сертификатов/грамот, отчёт по протоколу.
- **Администратор** — создание и редактирование учётных записей преподавателей,
  заполнение карточек олимпиад, просмотр и публикация подготовленных протоколов,
  годовой отчёт.

## Аутентификация

Вход выполняется **по логину и паролю** (стандартные формы Django `LoginView`).
Регистрация студентов — через форму `/register/`.

## Архитектура

Классическая структура Django (MVT):

| Слой | Где |
|------|-----|
| **Model** | `accounts/models.py`, `olympiads/models.py` |
| **View** | `accounts/views.py`, `olympiads/views.py` |
| **Template** | `templates/` |
| Маршруты | `*/urls.py`, `config/urls.py` |
| Формы | `*/forms.py` |
| Контроль доступа | `olympiads/decorators.py` (проверка ролей) |

- `config/` — настройки проекта
- `accounts/` — пользователи с ролями и профили
- `olympiads/` — олимпиады, записи, протоколы, результаты, сертификаты

## Запуск

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_demo      # демо-данные и тестовые учётки (опционально)
python manage.py runserver
```

Откройте http://127.0.0.1:8000/

### Тестовые учётные записи (после `seed_demo`)

| Роль | Логин | Пароль |
|------|-------|--------|
| Администратор | `admin` | `admin123` |
| Преподаватель | `teacher` | `teacher123` |
| Студент | `student` | `student123` |

Без demo-данных создайте администратора: `python manage.py createsuperuser`.

## Отчёты

- Протокол по олимпиаде — `/report/protocol/<id>/` (CSV для Excel).
- Годовой отчёт — `/report/annual/<год>/` (CSV для Excel).
